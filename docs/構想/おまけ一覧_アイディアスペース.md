# おまけ一覧・アイディアスペース

## 目的

このファイルは，ゲーム拡張の思いつきを一時保存するためのメモです．

## アイディア

- 片手練習モード（右手のみ／左手のみ）
- 小節ごとのループ練習
- テンポ可変練習（50%〜110%）
- ミスした箇所のみ自動リトライ
- 指番号（1-5）ガイド表示
- 鍵盤ハイライト履歴（直近3秒）
- 練習ログ保存（日時，曲，精度，ミス回数）
- ランク別ミッション（例: Accuracy 85%以上）
- コード練習モード（和音分解，転回形）
- 学習向けコメント表示（この小節は右手の跳躍に注意，など）

## 保留中の課題

- PCキーボードと88鍵ピアノの対応差の扱い
- ロングノーツと実際のペダル挙動の違い
- 音源ライセンス管理の運用ルール
