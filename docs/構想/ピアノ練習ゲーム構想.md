# ピアノ練習ゲーム構想

## ゴール

現在の斜めレーン型音ゲーを発展させ，PCキーボードでピアノ運指練習ができるWebアプリへ拡張する．

## 方向性

- 斜めレーン表現は維持する．
- 下部に3D風の白鍵・黒鍵UIを配置する．
- ノーツは鍵盤列に対応して落下する．
- 押下時に対応鍵盤が発光し，サンプル音源またはMIDI音源を鳴らす．

## 実装フェーズ案

1. 鍵盤UI導入

- まずは2オクターブ（24〜29鍵）で開始する．
- 白鍵と黒鍵のレイヤーを分離し，押下時の見た目差をつける．

2. キー割当導入

- ホームポジション周辺のキーを優先して割り当てる．
- 白鍵優先モードと黒鍵含むモードを切り替え可能にする．

3. 楽譜入力導入

- MusicXMLまたはMIDIを読み込み，音高と長さをノーツへ変換する．
- 最初は単旋律のみ，次に和音へ対応する．

4. 練習支援導入

- 小節ループ，テンポ変更，片手練習，採点履歴を追加する．

## 最終イメージ

- YouTubeのピアノ落下動画に近い視覚性を持つ．
- ただし動画ではなく，実際に入力して練習できるインタラクティブ形式にする．
- 学習補助として，楽譜から自動生成した練習譜面を提供できる状態を目指す．
