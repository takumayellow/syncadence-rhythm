## 進行中: スマホ対応（モバイルフレンドリー化）

> ブランチ: `claude/mobile-friendly-app-Fcrz9`

### 現状の課題

- `.lanes` の `clip-path` でタッチ領域が台形に切り取られているが `touch-action` 未設定のためスクロールと競合する
- `body { overflow: hidden }` + `min-height: 100vh` がモバイルブラウザのアドレスバー伸縮で崩れる
- セッティングパネルが小画面でスクロールできず内容が切れる
- ボタンのタッチターゲットが小さく誤タップしやすい
- ポートレート（縦向き）での playfield 高さが不適切
- ステータス欄・曲選択欄が狭い画面で窮屈
- キーボード操作の説明文がモバイルでも表示されている
- ランドスケープ（横向き）時の最適化が未対応

### 対応内容

- [ ] `touch-action: none` をレーンボタンに追加（スクロール干渉防止）
- [ ] `touch-action: manipulation` をコントロールボタンに追加（高速タップ対応）
- [ ] `min-height: 100dvh` でモバイルブラウザのアドレスバー伸縮に対応
- [ ] セッティングパネルを `overflow-y: auto` + `max-height` でスクロール可能に
- [ ] ボタンの最小タッチターゲットを 44px に
- [ ] モバイル向けブレークポイントの改善（480px 以下）
- [ ] 判定ライン位置をモバイルで調整（`--judge-line-bottom` を小さく）
- [ ] ランドスケープ向けのメディアクエリ追加
- [ ] キーボード説明文をモバイルでは非表示に
- [ ] `viewport` に `user-scalable=no` を追加（ゲーム中の誤ズーム防止）
- [ ] 結果画面にリスタートボタンを追加
