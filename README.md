# プロセカ風リズムゲーム（ブラウザ版デモ）

このフォルダは独立したGitリポジトリとして初期化済みです．

## 起動方法

1. `index.html` をブラウザで開く．
2. `START` を押す．
3. `D / F / J / K` キー，または各レーンをタップしてノーツを叩く．

## 実装内容

- 4レーン落下ノーツ
- 判定: `Perfect / Great / Good / Miss`
- スコア，コンボ表示
- 30秒の固定シード譜面（毎回同じ譜面）
- PCキーボード入力とモバイルタップの両対応
